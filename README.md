#  Unemployment Analysis Using Python

##  Project Overview

This project analyzes unemployment data to identify trends, patterns, and regional variations. Using Python and data visualization techniques, it explores unemployment rates over time, helping to uncover meaningful insights for data-driven decision-making.

##  Objective

* Analyze unemployment trends across different regions and time periods.
* Perform data cleaning and preprocessing.
* Visualize unemployment patterns using charts and graphs.
* Generate insights from the analyzed data.

##  Dataset

The dataset contains unemployment-related information, including:

* Region/State
* Date/Time
* Estimated Unemployment Rate (%)
* Estimated Employed Population
* Estimated Labour Participation Rate (%)

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

##  Exploratory Data Analysis (EDA)

* Checked dataset structure and missing values.
* Cleaned and preprocessed the data.
* Generated summary statistics.
* Explored relationships between variables.

##  Data Visualization

The project includes:

* Line Charts
* Bar Charts
* Heatmaps
* Histograms
* Box Plots

These visualizations help identify unemployment trends, regional differences, and changes over time.

##  Key Insights

* Identified regions with high and low unemployment rates.
* Analyzed changes in unemployment over time.
* Compared employment and labour participation across regions.
* Highlighted important trends using visualizations.

##  How to Run

1. Clone this repository.
2. Install the required libraries:

   ```bash
   pip install -r requirements.txt
   ```
3. Open `Unemployment Analysis.ipynb` in Jupyter Notebook.
4. Run all cells to reproduce the analysis and visualizations.
